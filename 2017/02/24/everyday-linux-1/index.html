每天一个Linux命令(1)

2017.02.24 Chen Linux  热度 ℃ 条评论

shell 下快捷键

光标移动命令

  • Ctrl + a
    移动到行首
  • Ctrl + e
    移动到行尾
  • Ctrl + f
    向前移动
  • Ctrl + b
    向后移动
  • Alt + f
    按单词前移

  • Alt + b
    按单词后移

    删除命令

  • Ctrl + u
    从光标处删除至命令行首
  • Ctrl + k
    从光标处删除至行尾
  • Ctrl + w
    从光标删除至字首
  • Ctrl + d
    删除光标处的字符

  • Ctrl + h
    删除光标前的字符

    重新显示命令

  • Ctrl + r
    逆向搜索命令历史
  • Ctrl + g
    退出搜索模式
  • Ctrl + p
    显示上一条命令

  • Ctrl + n
    显示下一条命令

    控制命令

  • Ctrl + l(L)
    清屏
  • Ctrl + s
    阻止屏幕输出
  • Ctrl + q
    允许屏幕输出
  • Ctrl + c
    终止命令
